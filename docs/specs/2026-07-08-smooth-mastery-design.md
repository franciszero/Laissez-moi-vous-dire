# 设计备用：平滑证据权重掌握度（未实现，用户拍板暂缓）

日期：2026-07-08。状态：**已设计、已验算、未实现**（用户选择先不动代码）。

## 动机

用户直觉：色块应平滑表示「一个词被记住的程度」，同一天第一次/第二次作答的贡献
本质不同，需要数值定义。现行 `mastery.mastery_score` 的「当天第一次为准」是硬开关：
同日第二次权重=0、跨午夜突变回 1（午夜悬崖）。真实案例：avoir l'air 当天
19:35 错 → 19:38 对，色块纹丝不动，用户以为没记上（其实 ✅ 与 DB 都记了）。

## 数值定义

每次作答都是观测，权重 = 它证明的记忆间隔 × 现有半衰期衰减：

    w_i = s(Δt_i) × 0.5^(age_i / 14天)
    s(Δt) = Δt / (Δt + τ)          # 间隔因子；τ = 1 天（可调）
    第一次接触（无前次）s = 1        # 最冷，无先验曝光
    掌握度 = WilsonLower(Σ w_i·ok_i, Σ w_i)   # z=1.28 不变

Δt = 距上一次接触该词(同技能)的时间（任何作答/偷看都算接触）。
语义一句话：**「答对」的价值 = 它证明你记住了多久**。

## 验算（用真实 DB 数据，脚本见 git log 本文件的提交说明）

| 场景 | 现行 | 平滑(τ=1d) |
|---|---|---|
| 当天 错→3分钟后对 | 0.000 | 0.000 |
| +明天冷启动对 | 0.169 | 0.071 |
| 今晚每小时刷4次全对 | 0.000 | 0.009（刷不绿） |
| 隔1/2/4天冷启动3次对 | 0.610 | 0.560 |
| 新词第一次就对 | 0.379 | 0.379（不变） |
| 看答案→39秒照抄 | 0.000 | 0.000 |

好处：无午夜悬崖；同日重试有连续定义；防刷分从硬规则变成数学性质
（间隔小→权重小），「当天第一次为准」可整条删除。
代价：比现行更保守（隔 1 天的成功只算 0.5 个观测），全表颜色整体略变淡、
绿得更慢；τ=0.5 天可调暖（1天间隔→0.67）。
